CSV Format Guide
Required CSV Columns:
Column Description Example Required
sip_name Name for your SIP HDFC Top 100 Monthly SIP ✅ Yes
fund_name Mutual fund name (will be matched automatically) HDFC Top 100 Fund - Growth ✅ Yes
investment_date Date of investment (YYYY-MM-DD) 2024-01-15 ✅ Yes
amount Investment amount 5000 ✅ Yes
nav_price NAV price on investment date 650.75 ✅ Yes
units Units allocated (calculated if blank) 7.6847 ❌ Optional
frequency SIP frequency monthly ❌ Optional
transaction_id Bank/AMC transaction reference TXN123456789 ❌ Optional
Sample CSV Content:
sip_name,fund_name,investment_date,amount,nav_price,units,frequency,transaction_id

Important Notes:
  • Fund Matching: Fund names will be matched against our database. Use exact names from the SIP creation page.
  • Date Format: Use YYYY-MM-DD format for dates (e.g., 2024-01-15)
  • NAV Prices: Use historical NAV prices from your statements for accurate calculations
  • Multiple SIPs: You can include multiple SIPs in the same CSV file
  • Auto-Creation: If a SIP doesn't exist, it will be created automatically
