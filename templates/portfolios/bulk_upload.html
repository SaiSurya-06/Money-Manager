Bulk Upload Holdings

Upload holdings from a CSV file to quickly populate your portfolio

CSV Format Requirements

Your CSV file should have the following columns (header row required):

  • symbol - Asset symbol (e.g., AAPL, RELIANCE.NS)
  • name - Asset name (optional)
  • quantity - Number of shares/units owned
  • average_cost - Average price per share/unit
  • asset_type - Type: stock, mutual_fund, etf, bond (optional, defaults to stock)
  • current_price - Current market price (optional)

Upload Instructions:
  1. Select the portfolio where you want to add the holdings
  2. Choose your CSV file (must be properly formatted)
  3. Click "Upload Holdings" to process the file
  4. Review any error messages and correct your CSV if needed
  5. Successfully imported holdings will appear in your portfolio
Common Issues:
  • Make sure your CSV file has a header row with the correct column names
  • Use decimal numbers for quantity and prices (e.g., 100.50, not 100,50)
  • Asset symbols should be unique within the same portfolio
  • If an asset with the same symbol already exists, it will be updated
