Upload Instructions
Supported File Types:
  • Excel (.xlsx, .xls)
  • CSV (.csv)
  • PDF (.pdf) - Bank statements
Maximum file size: 10MB
Required Columns (CSV/Excel):
  • Date: Transaction date (YYYY-MM-DD)
  • Description: Transaction description
  • Amount: Transaction amount (positive numbers only)
  • Type: 'income' or 'expense' (lowercase)
PDF Bank Statements:
  • Format: Text-based PDF (not scanned images)
  • Content: Transaction tables with date, description, amounts
  • Note: System will auto-detect transaction type
  • Limitation: May not work with all bank formats
Optional Columns (CSV/Excel):
  • Category: Category name (will be created if not exists)
  • Notes: Additional transaction notes
Note: The account is selected above, not in the file. All transactions will be assigned to the selected account.
